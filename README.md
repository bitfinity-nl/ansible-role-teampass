## Description
Installing TeamPass server on Ubuntu

## Requirements
- Ubuntu 18.04
- MySQL server 5.7.x


## Role Variables

  MySQL_ROOT_ACCOUNT  : root

  MySQL_ROOT_PASSWORD : password

  tp_dbname           : teampass

  tp_dbuser: teampass : teampass

  tp_dbpass: teampass : teampass
